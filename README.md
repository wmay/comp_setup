Dotfiles focused on Ubuntu, data science, and Emacs.

# Emacs

## Language mode packages

- csv-mode
- cython-mode
- ess (R/S+/Stata/etc.)
- markdown-mode
- poly-R (Rmarkdown)
- sql-indent (decent SQL indentation)
- stan-mode
- web-mode

## Installation

```sh
emacs --script install_packages.el
```
